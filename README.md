# i3 dotfiles

![rice](preview.png)

my daily driver arch + i3 setup

## whats in here

- **wm**: i3
- **bar**: polybar
- **launcher**: rofi
- **terminal**: alacritty
- **shell**: bash + starship
- **compositor**: picom
- **notifications**: dunst
- **file manager**: thunar
- **colors**: pywal
